# i18n — Pack It!

## Languages
- English: built-in
- Chinese (Simplified) and Hindi: on-demand via translation API

## Game-specific translation keys

All question texts use `{placeholder}` substitution so they translate cleanly.

```json
{
  "l1_question": "Pack {total} {items} equally into {groups} {containers}.",
  "l2_find_total": "Each {container} holds {unit} {items}. You have {groups} {containers}. How many {items} in total?",
  "l2_find_groups": "There are {total} {items}. Each {container} holds {unit}. How many {containers} do you need?",
  "l3_find_groups": "{totalA} {items} filled {groupsA} {containers} equally. How many {containers} are needed for {totalB} {items}?",
  "l3_find_total": "{totalA} {items} filled {groupsA} {containers} equally. How many {items} fill {groupsB} {containers}?",
  "l4_find_unit": "{total} {items} were shared equally among {groups} {containers}. How many {items} per {container}?",
  "round_load": "Load",
  "round_pack": "Pack",
  "round_ship": "Ship",
  "unit_rate_badge": "{unit} {items} per {container}",
  "feedback_correct": "✓ {answer} {unit}",
  "feedback_wrong": "Not quite — try again",
  "answer_placeholder_items": "{items}",
  "answer_placeholder_containers": "{containers}",
  "game_complete_title": "All packed and shipped!",
  "rotate_prompt": "Please rotate your device for the best experience 🔄"
}
```

## Grouping pair names (must be translated per language)
Each `GroupingPair` has `item`, `itemPlural`, `container`, `containerPlural` — all translatable. The translation API receives the English values and returns localised equivalents.

## Blackboard text
Blackboard steps use numbers and mathematical notation (`→`, `×`, `÷`, `=`) — these are not translated. Only the item/container words within the steps are localised.

## Static vs substitution
All question prompts use `{placeholder}` substitution. This means:
- Translation API receives the template string once per language
- Substitution happens at runtime
- No per-question translation API calls
